### **Réseau :** 

**PCAP is a Wireshark file which allows you to analyze the flow of network traffic, inspect individual packets, identify protocols. When you send a request, packets are created that travel across the network and arrive at the destination, where they are reassembled and processed. So you can analyze each of them to see if a flag is there for exemple in a  login request. 

UDP: User Datagram Protocol is a network protocol that sends data as individual packets without establishing a connection or guaranteeing delivery.

TCP (same as HTTP): Transmission Control Protocol is a network protocol that establishes a connection to reliably send and receive data in order between devices.
